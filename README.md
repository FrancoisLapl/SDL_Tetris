Petit jeu de tetris gossé à la main et écrit en C et utilisant la librairie SDL pour les aspect graphique, son et gestion du clavier.

#Pre-requis</br>
-librairie SDL2</br>
-make</br>
-gcc(Peut changer, mais non testé)</br>
-objdump(optionnel)</br>
